## 📝 Project Overview

A fully responsive Coca-Cola brand landing page with live weather API integration, interactive maps, and real-time consumption counter.

## ✨ Features

- 🌤️ **Live Weather** - Real-time weather for 6 global Coca-Cola locations
- 🗺️ **Interactive Map** - Leaflet.js map with weather popups
- 📊 **Live Counter** - Real-time consumption statistics
- 🌍 **Multi-language** - 4 language support (EN, ES, DE, HI)
- 🎨 **Modern Design** - Tailwind CSS with custom animations
- 📱 **Fully Responsive** - Works on all devices
